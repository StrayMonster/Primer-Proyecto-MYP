Bienvenido a Weathering, la página para visualizar el clima.
Necesita seguir los siguientes pasos para la correcta ejecución del programa.

Primero, se usó Visual Studio Code para crear y ejecutar el programa. Por lo que se puede instalar en su sitio oficial aquí: https://code.visualstudio.com/
O en el sitio de Microsoft: https://visualstudio.microsoft.com/es/

Segundo, se necesita tener python instalado, puede ser descargado en su sitio oficial: https://www.python.org/downloads/

Tecero, una vez descargado python, ejecutar este comando en su terminal del sistema: pip install Flask python-dotenv pandas requests
Si eres usuario de windows, busca en la barra de busqueda: "Símbolo de sistema".

Cuarto, deberás tener una key de la api de open weather, si no la tienes registrate en el sitio oficial: https://openweathermap.org/api
Puedes saltarte este paso si ya tienes una e ir al paso 5.

Quinto, en el archivo llamado: ".env.extend" deberás colocar tu clave de api donde se indica ahí. Finalmente, eliminar la
extensión: ".extend" y dejar solo ".env".

Sexto, en la barra de herramientas que se encuentra en la parte superior de VSC. Darle click al cuadrado hecho de 4 cuadrados, dice extensions.
En este apartado buscar: "Python", debes seleccionar el primer resultado que es por Microsoft. Posteriormente instalarlo.

Septimo, una vez hecho lo anterior dirigirte al archivo Público.py o Vuelo.py Dependiendo que persona lo vaya a usar.
Si es parte del público en general, ubicarse en el archivo Publico.py. Para ver como funciona con el programa para este público.
Si es parte del personal de vuelo, ubicarse en el archivo Vuelo.py Para ver como funciona el programa para este público. 

Octavo, en la parte superior derecha, se va a encontrar un triangulo con la punta en horizontal, con una cuña invertida.
Deberá darle click a la cuña invertida, ahí se desplegará un submenú, se deberá seleccionar la opción: "Run Python File in Dedicated Terminal".

Noveno, se desplegara la terminal, en la que saldrá un texto en rojo y finalmente una dirección http.
Colocar el cursosr en dicha dirección y presionar la tecla control. En ese momento podrás seleccionar la liga.
Darle click y te enviará a la página web, en la cual deberás seguir las instrucciones de la misma.

Decimo, una vez que haya acabado de usar el programa, en la misma terminal donde ejecuto el programa, presionar la tecla control + c para finalizar la ejecución.

Nota: La modificación del código por el usuario que disponga de esto, puede causar errores y que no se ejecute como es debido.
Es responsabilidad del acreedor de estos elementos, usarlo con responsabilidad.
